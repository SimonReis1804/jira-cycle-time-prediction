# src/optimization

Dieser Ordner enthält Funktionen zur Ausführung der Hyperparameteroptimierung

## Inhalt
- Hyperparameteroptimierung mit Random Search
- Zwei verschiedene Hyperparameteroptimierung Ansätze

## Hinweis
Ausführung der Hyperparameteroptimierung:
```bash
python -m src.optimization.hyp_opt_jira <Modell>
python -m src.optimization.hyp_opt_time_based <Modell>
```
